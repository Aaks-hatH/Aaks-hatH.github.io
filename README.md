# Terminal-style About Page — aaks-hatH

Simple static site: a terminal-like About page for **Aakshat Hariharan (aaks-hatH)**.

## Files
- `index.html` — main page
- `assets/css/style.css` — styles
- `assets/js/main.js` — interactive commands and text
- `README.md`, `LICENSE`, `.gitignore`

## Edit content
Open `assets/js/main.js` and edit the `HANDLER` object to change what each command shows (e.g., `about`, `skills`, `projects`).

## Deploy to GitHub Pages
1. Create a repo on GitHub (for example `portfolio-term`).
2. Push these files to the repo.
3. In the repo Settings → Pages, set the source to the `main` branch (root). Save.
4. Your site will be available at `https://Aaks-hatH.github.io/<repo>/`.

If you want the site to be at `https://Aaks-hatH.github.io` (root user page), name the repo exactly `Aaks-hatH.github.io`.
